Selected work

SE3 Electricity Price Forecast

A live, end-to-end system that forecasts Swedish electricity prices (SE3 zone) 24 hours ahead with calibrated uncertainty bands. Built, deployed, and retrained automatically on GCP.

LightGBM quantile model (p05/p50/p95) with 78 engineered features. Hourly ingestion from ENTSO-E and Open-Meteo via Cloud Run Jobs, DuckDB on GCS, weekly retraining via Cloud Scheduler, MLflow for versioning. A LangGraph agent answers plain-English questions over the live data with cited sources.

graph-rag-memory

A graph-augmented RAG memory system with knowledge-graph retrieval and write-time deduplication, running fully on a local LLM.

Tiered source and derived memory on ChromaDB with decay-based compaction and cosine plus novelty deduplication. NetworkX knowledge graph for multi-hop retrieval. No external APIs.
